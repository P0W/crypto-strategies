# Content Directory

This directory contains markdown files for external communication, blog posts, and social media announcements.

## Structure

### `social-media/`
Platform-specific announcements and launch materials:
- **reddit_r_rust.md** - Technical post for r/rust community
- **reddit_r_algotrading.md** - Trading results focus for r/algotrading
- **hackernews.md** - "Show HN" post format
- **README.md** - Launch strategy and timeline

### `blog/`
Long-form content for dev.to, Medium, and personal blog:
- Coming soon: Deep-dive articles on architecture, strategy design, and performance

## Usage

These files are templates ready for posting on respective platforms. Review and customize before posting to match your voice and current metrics.
